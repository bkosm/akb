# Security Policy

## Reporting a Vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Instead, use one of the following:

- **GitHub private vulnerability reporting**: [Report a vulnerability](https://github.com/bkosm/akb/security/advisories/new)
- **Email**: bartoshmen@gmail.com

Include a description of the issue, steps to reproduce, and any relevant version information. You will receive a response within 7 days.
